This plugin provide you facility to search scholarships all around the World.

=======Description==========

This plugin provide you facility to search scholarships all around the World. You can search your suitable scholarships by using different parameters and filters provided by this plugin. Very smooth searching and pagination is provided by this plugin to display the scholarship records.

=======Installation=========

1. Upload the plugin to the "/wp-content/plugins/" directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings -> "WP Scholarship Setting" screen to make some setting of scholarship search form.
4. The scholarship form will be accessible by following shortcode [wp_education_scholarship_search]. You can place this shortcode on any wordpress page and enjoy the features of Scholarship plugin.

=======Requirements========

CURL: For scholarship search this pluign uses some API's. So site must support CURL to function this plugin properly.
This plugin uses JS files for smooth search and pagination. So make sure that wp_head() & wp_footer() are including in your activated theme's header and footer files.

=======Services========

The scholarship search of this plugin is relying on 3rd party services. In this plugin 3rd party API's are using to get scholarship search results. This plugin never store any scholarship data on installed sites, It always fetch scholarship data using 3rd party services by using API's.

=======Features============

1) Search scholarship all over the world.
2) Uses alot of fields to filter the scholarship data.
3) Review the Scholarship details from "View Details" link.
4) One can apply scholarship from link provided on scholarship detail page either by online or offline.
5) Provide shortcode [wp_education_scholarship_search] to use anywhere in wordpress.
6) Support searching on Mobile devices too.
7) You can customize the limit of results showing per page from backend settings. You can also change the heading of scholarship search form and "Search" button from backend settings.
